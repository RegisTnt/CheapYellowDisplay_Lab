# Vérifications prévues

- parsing Open-Meteo complet et incomplet ;
- parsing strict de `ferme` et `ouvert` ;
- absence de retry sur `/pieton` et `/voiture` ;
- verrouillage pendant l'envoi et pendant trois secondes ;
- conservation de la météo après perte Wi-Fi ;
- respiration du voyant après trente secondes d'ouverture ;
- scénarios `DEMO_SCENARIO` 0 à 5 ;
- calibration des quatre coins de la dalle avant usage quotidien.
